Google's AI Ecosystem

Ever got confused with different google apps and models, where to go for what, what are usable, what is available today?

Lets try to understand what they can do and debunk few common assumptions on what I cant do.

Google NotebookLM

Web & Mobile App Note Taker

What is it?

A personal research assistant designed to help you synthesize information and consolidate your learning.

What it can do:

  • Add over 200 sources, from PDFs and website URLs to YouTube videos.
  • Chat with your sources to extract information and insights (similar to RAG).
  • Generate a podcast-style dialogue between two hosts based on your material.
  • Create useful outputs like timelines, study guides, and FAQ documents.
  • Summarize entire sources and ask specific questions about the content.
  • Define and customize the AI's conversational style.

What it can't do:

  • It cannot access the internet for general knowledge; it is strictly limited to your provided sources.
  • It lacks a text-to-speech feature to read content aloud.

Gemini

Web & Mobile App Chatbot

What is it?

A versatile, general-purpose conversational AI, similar to ChatGPT.

What it can do:

  • Generate a wide range of content, including text, video, and software code.
  • Utilize the "Canvas" experience to instantly preview generated code.
  • Perform in-depth research on a variety of topics.
  • Ground its responses by uploading and referencing your own documents.

What it can't do:

  • It cannot directly edit an existing image. However, you can describe the desired changes, and it will attempt to generate a new, modified image.

Google AI Studio

Web App Prototyping

What is it?

A comprehensive, web-based platform for prototyping with Google's generative AI models.

What it can do:

  • Build and test conversational agents similar to Gemini.
  • Generate multimodal content, including audio, video, text, and images.
  • Facilitate the creation of audio-to-audio dialogues.
  • Allows for screen sharing during conversational interactions.
  • Build simple applications using pre-made templates or starting from scratch.
  • Provides a library of prompt templates to get started quickly.
  • Easily obtain API keys for integrating Gemini models into your own projects.
  • Access advanced settings for tool calling, structured output, safety, and grounding.

What it can't do:

  • It does not have the capability to perform automated browser actions.

Gemini Live

Mobile App Audio/Video Chat

What is it?

An interactive, real-time audio and video conversation with an AI chatbot.

What it can do:

  • Engage in a real-time, fluid conversation.
  • Stream your screen for interactive assistance.
  • Read documents aloud to you.
  • Use your camera to show what's around you and ask questions.

What it can't do:

  • It is not a direct mobile equivalent of the developer-focused Google AI Studio.

Google Labs

Web App Experimental

What is it?

A platform for trying out early-stage, experimental features from Google.

What it can do:

  • Whisk: Create videos from images.
  • Flow: Automate the video creation process for potentially full-length videos.
  • Doppl: Virtually try on clothing and accessories.

What it can't do:

  • It is not a platform for creating new AI models.

Google Vertex AI Agent Builder

Web App (GCP) Agent Builder

What is it?

A low-code/no-code platform on Google Cloud for building enterprise-grade AI agents.

What it can do:

  • Offers numerous tool integrations for enterprise systems.
  • Includes an "Agent Engine" to run production-level agents at scale.
  • Provides an "Agent Space" which acts as a registry for enterprise agents.
  • Features an "Agent Garden" with a collection of ready-made agents.

What it can't do:

  • Does not allow for deploying agents using only natural language commands.

Google ADK

Agentic Package

What is it?

An Agent Development Kit (ADK) for building complex agentic systems.

What it can do:

  • Develop sophisticated multi-agent systems using code.
  • Integrates with MCP (Multi-platform Control) and tool calling.
  • Provides built-in session management and memory for agents.
  • Offers authentication, a lightweight UI, and an agent API out of the box.
  • Includes an Agent-to-Agent (A2A) protocol for inter-agent communication.

What it can't do:

  • It does not come with a library of readily available, pre-built agents.

Gemini for Chrome

Extension Browser Assistant

What is it?

An AI-powered assistant integrated directly into the Chrome browser.

What it can do:

  • Provides assistance without needing to switch tabs.
  • Can directly access and analyze the content of the current page.

What it can't do:

  • Lacks the full capabilities of advanced browser agents like OpenAI's Operator.
  • Cannot yet perform browser actions (like clicking or typing) on the user's behalf.

Project Mariner

Agent Browser Agent

What is it?

A next-generation browser agent focused on human-agent interaction.

What it can do:

  • Automate complex, multi-step tasks across different websites.
  • Intended to match the capabilities of agents like OpenAI's Operator.
  • Will feature native integrations with all Google applications.

What it can't do:

  • It is not yet generally available (GA) and is limited to US users with the Google One AI Premium plan (formerly Ultra).

Gemma Models

Model

What is it?

A family of low-latency, multimodal language models designed for specific problems.

What it can do:

  • Offers a range of low-latency models for developers to build applications.
  • Includes specialized open-source models like Sign Gemma (sign language) and Med Gemma (medical data).
  • Available on platforms like Hugging Face (https://huggingface.co/collections/google/).
  • Can be used within Google AI Studio for experimentation.

What it can't do:

  • They are foundational models, not general-purpose chatbots.

Gemini in all google apps

Web & Mobile Apps Extension

What is it?

An integrated extension that brings Gemini's power into all Google apps.

What it can do:

  • Leverage Gemini models within daily apps like Gmail, Drive, YouTube, and Photos.
  • Now also available for use directly within Google Colab notebooks.

What it can't do:

  • Cannot perform actions without explicit user permissions and inputs.

Google AI mode

Browser Native Feature

What is it?

The AI-powered conversational search feature within the main Google Search tab.

What it can do:

  • Provides a conversational search experience with follow-up questions.

What it can't do:

  • It does not remember conversations across different tabs or windows.
  • It is not a full-fledged agentic browser.

Firebase Studio

Web App

What is it?

A web-based platform that helps you build full-stack applications with AI assistance.

What it can do:

  • Import an existing repository or create a new app from scratch.
  • Supports writing code in multiple programming languages.
  • Offers a "Canvas" experience for live previews (more capable than Gemini's HTML-only canvas).
  • Publish your applications directly from the platform.
  • The "Code view" opens a complete online VS Code editor.

What it can't do:

  • It is not a general-purpose IDE code assistant.
  • It is not an assistant bot for Firebase-specific questions.

Gemini Code Assist

Extension Code Assistant

What is it?

An AI assistant designed to accelerate the software development lifecycle.

What it can do:

  • Provides an IDE extension to help with coding issues directly in your editor.
  • Offers a Command Line Interface (CLI) for building agents, similar to OpenAI Codex.
  • Interacts with MCP (Multi-platform Control) servers.
  • Assists with Code Pull Request (PR) reviews and other GitHub actions.
  • Integrates with the Google Cloud Shell editor.

What it can't do:

  • It is not yet optimized for working with codebases that span multiple files.
